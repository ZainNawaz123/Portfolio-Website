FPGA Random Maze Generator & Solver

Verilog (DE1-SoC)

Designed and implemented a fully hardware-based random maze generator, solver, and VGA renderer in Verilog on a DE1-SoC FPGA (no CPU/software)

  • Built a dual-port BRAM memory interface to support maze generation (write) and VGA rendering (read)
  • Implemented DFS backtracking logic with wall-state and visited-cell tracking to generate perfect mazes
  • Developed a VGA rendering pipeline to display maze walls, start/end markers, and an overlaid solution path
  • Resolved hardware timing and randomness bugs, eliminating infinite-generation and VGA alignment issues
